\textbf{Lemma 1 (Monotonicity of Acyclicity):}
If $\text{Acyclic}(r_2)$ and $r_1 \subseteq r_2$, then $\text{Acyclic}(r_1)$.

\textbf{Proof sketch:}
Since transitive closure is monotone, $r_1 \subseteq r_2$ implies $r_1^+ \subseteq r_2^+$.
If $r_2^+$ is irreflexive (i.e., $\text{Acyclic}(r_2)$), then any subset $r_1^+$
must also be irreflexive, hence $\text{Acyclic}(r_1)$.
\textit{(Formally verified in Lean 4; see [reference to your code]).}
\qed

\textbf{Theorem:} If SC is acyclic, then TSO is acyclic.

\textbf{Proof:}
We have $\text{TSO} \subseteq \text{SC}$ because:
\begin{itemize}
  \item $\text{po} \cap ((R \times W) \cup (W \times W)) \subseteq \text{po} \subseteq \text{SC}$
  \item $\text{co} \subseteq \text{SC}$, $\text{fr} \subseteq \text{SC}$
  \item $\text{rf} \cap \text{external} \subseteq \text{rf} \subseteq \text{SC}$
\end{itemize}
By Lemma 1, $\text{Acyclic}(\text{SC})$ implies $\text{Acyclic}(\text{TSO})$.
\textit{(Formally verified in Lean 4; see [reference]).}
\qed
